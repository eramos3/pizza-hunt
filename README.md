# pizza-hunt

## Description
A social media recipe app where users can post their creative spins on pizza, add comments to recipes, and create threads within comments.

## Setup
```
git clone git@github.com:eramos3/pizza-hunt.git
```

## Built with 
 * Express.js
 * MongoDB
 * Mongoose
